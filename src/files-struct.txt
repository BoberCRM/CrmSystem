/app: Основная директория для страниц и маршрутов, начиная с Next.js 13.
/assets: Статические файлы, такие как изображения и иконки.
/components: Переиспользуемые React-компоненты.
/contexts: Контексты для глобального состояния, например, для аутентификации.
/hooks: Пользовательские хуки для общих функций.
/services: Сервисы для API-запросов и других операций.
/store: Хранилище для глобального состояния приложения.
/utils: Вспомогательные функции и утилиты.
/middleware: Функции мидлвейров для обработки запросов или выполнения логики до рендеринга.
/public: Публичные файлы, доступные напрямую по URL.
/pages: Страницы приложения в старом стиле, используемые если не используется директория app.
